가이드: 깃헙(GitHub)에서 개인용 접근 토큰 생성

윤아 2022-01-06
  • 가이드
  • GIT
  • 첫주

안녕하세요, 수강생 여러분!

2021년 8월 13일부터 깃헙(GitHub)에서 Git 작업을 인증할 때 더 이상 계정 비밀번호를 이용할 수 없습니다. 그 대신 토큰 기반 인증을 사용해야 한다고 하는군요. 그래서 이번 포스팅에서는 토큰 기반 인증 중 하나인 **개인용 접근 토큰(PAT, Personal Access Token)**을 사용하는 방법을 설명드리겠습니다.

이 포스트에서는 Windows 기반의 PC에서 Chrome 브라우저를 사용할 때를 기준으로 설명합니다.

  1. GitHub 사이트에 로그인합니다.

  2. 페이지 우측 상단의 프로필 사진을 선택하면 나오는 메뉴에서 Settings를 선택합니다.

  3. 페이지 좌측 사이드바에 있는 Developer settings를 클릭합니다.

  4. 왼쪽 사이드바에 있는 Personal access tokens를 클릭합니다.

  5. Generate new token을 클릭합니다.

  6. Note에는 토큰을 설명하는 내용을 적습니다. 어떤 토큰인지 정확하게 적으면 나중에 헷갈리지 않겠죠^^? (ex. POCU labs/assignments)

  7. Expiration에서는 토큰 만료 기간을 설정합니다. 기본값은 30일이지만 우리는 POCU 아카데미 학기 종료일을 만료 기간으로 설정해봅시다. 드롭다운에서 Custom...을 클릭하여 학기 종료 이후 시간으로 설정합니다.

  8. Select Scopes에서는 토큰으로 접근할 수 있는 것들을 결정합니다. 우리는 repo만 선택하면 충분합니다.

  9. 이제 모든 준비가 끝났습니다. Generate token 버튼을 클릭하면 토큰이 생성됩니다.

  10. 아참참! 만일을 대비하여 생성된 토큰을 즉시 안전한 곳에 복사해둡니다. 이 페이지를 벗어나면 더이상 토큰을 확인할 수 없으니 주의하세요!
