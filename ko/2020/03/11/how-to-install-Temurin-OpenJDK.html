가이드: Temurin OpenJDK 설치하기

윤아 2020-03-11
  • 첫주
  • 가이드

안녕하세요, POCU 수강생 여러분! 윤아입니다 :D

이번 시간에는 Adoptium 그룹이 제공하는 Temurin OpenJDK 버전 11을 설치하는 방법을 알아봅시다.

이 설치 방법은 POCU 아카데미의 과목을 수강하시는 분들을 기준으로 작성되었음을 미리 알려드립니다.

  1. Adoptium 릴리스 페이지를 방문합니다.

  2. Version 드롭다운 메뉴에서 11을 선택합니다.

  3. 본인 컴퓨터에 적합한 jdk-11.0.XX.X+XXX 설치 파일을 다운로드합니다.

  4. 다운로드한 파일을 실행해서 설치를 시작합니다.

  5. 기본 값 설정을 사용해서 설치를 마무리합니다.

  6. 커맨드 창을 열어 java -version을 입력합니다. Java가 제대로 설치되었다면 커맨드 창에 아래 예와 같이 Java 버전이 출력됩니다.

    C:\Users>java -version
openjdk 11.0.21 2023-10-17
OpenJDK Runtime Environment Temurin-11.0.21+9 (build 11.0.21+9)
OpenJDK 64-Bit Server VM Temurin-11.0.21+9 (build 11.0.21+9, mixed mode)

매우 쉽죠? 그럼 즐거운 수업 되세요! :D
