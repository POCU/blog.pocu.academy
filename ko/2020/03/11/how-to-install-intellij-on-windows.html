가이드: IntelliJ IDEA 설치하기

윤아 2020-03-11
  • 첫주
  • 가이드

안녕하세요, POCU 수강생 여러분! 윤아입니다 ’-^d

이번 시간에는 Java 개발 통합 환경(IDE)인 IntelliJ IDEA를 설치하는 방법에 대해 알아보도록 합시다!

본 가이드는 IntelliJ IDEA 2023.3.1 버전을 설치합니다. IntelliJ의 버전에 따라 UI가 약간씩 다를 수 있습니다. 또한 이 설치 방법은 POCU 아카데미의 과목을 수강하시는 분들을 기준으로 작성되었음을 미리 알려드립니다.

  1. Temurin OpenJDK 설치 가이드를 따라 Temurin OpenJDK를 설치합니다.

  2. IntelliJ IDEA 홈페이지에서 커뮤니티 버전을 다운로드합니다.

  3. 다운로드한 설치 파일을 실행합니다.

  4. 설치 창에 나오는 설명에 따라 설치를 완료해주세요. 기본 값 설정을 사용하여 설치하면 됩니다.

  5. IntelliJ를 실행합니다.

  6. JetBrains의 개인정보처리방침(privacy policy)에 동의하라는 팝업이 등장하면 체크박스를 체크한 뒤 Continue를 누릅니다.

  7. 또한 IntelliJ 실행 도중 크래시가 날 경우, 진단(diagnostic) 데이터를 공유하겠냐는 팝업이 뜰 수도 있습니다. 굳이 동의하실 필요 없으니 Don't send를 선택한 뒤 계속 진행합니다.

  8. 이제 Temurin OpenJDK를 사용하도록 IntelliJ를 설정해야 합니다. 설정을 위해 먼저 프로젝트를 열어야 합니다. 새로 생성해주셔도 되고 기존에 있는 프로젝트를 Open을 통해 열어도 됩니다.

  9. 프로젝트를 열었으면 왼쪽 위 메뉴 바에서 File > New Projects Setup > Structure...를 선택합니다.

  10. Platform Settings 아래에서 SDKs를 선택합니다.

  11. SDKs 아래 Temurin OpenJDK가 나타난다면 **15번 (JDK 이름 변경)**으로 건너뛰세요.

  12. 제일 위에 있는 + 버튼을 선택한 뒤, Add JDK를 선택합니다.

  13. Temurin OpenJDK를 설치했던 디렉터리를 선택합니다.

    예) C:\Program Files\Eclipse Adoptium\jdk-11.0.16.101-hotspot.

  14. Ok를 선택합니다.

  15. JDK의 이름을 변경합니다. 본 가이드에서는 버전 11을 설치했으므로 Java-11이라고 이름을 변경했습니다.

  16. Project Settings 아래에서 Project를 선택합니다.

  17. 14번에서 추가했던 JDK를 프로젝트의 SDK로 선택합니다.

  18. Apply 와 OK를 선택합니다.

잘하셨어요! :D
