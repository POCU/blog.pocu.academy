가이드: 소스트리 설치

문서봇 2019-01-06
  • 첫주
  • GIT
  • 가이드

안녕하세요. POCU 아카데미 수강생 여러분

이 가이드는 Git GUI 툴인 소스트리(Sourcetree)를 설치하는 방법입니다.

  1. 소스트리 사이트에서 설치 파일을 다운로드하여 실행합니다.

  2. 계정 등록 화면이 나오면 건너뛰기를 선택합니다.

  3. 도구 설치 창이 나오면 Git 체크박스가 체크된 걸 확인하고, 고급옵션에서 기본적으로 줄 끝을 자동으로 처리하도록 설정(권장) 체크박스를 해제합니다.

  4. 이제 다음을 눌러서 설치를 마저 진행합니다

  5. Author Name, Email Address를 입력(자동 입력되는 경우도 있음)하고 설치를 마무리합니다.

  6. 설치 완료 후, SSH키를 불러오시겠습니까? 라는 메시지가 뜨면 아니오를 누릅니다.

소스트리 같은 툴은 실무에서 필수로 쓰이니 이번 기회에 배워놓으시면 나중에 도움이 많이 될 겁니다!

그럼 즐거운 강의되시길 바랍니다 :D
