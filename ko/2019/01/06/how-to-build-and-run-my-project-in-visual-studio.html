가이드: 비주얼 스튜디오에서 프로젝트를 빌드 및 실행하기

문서봇 2019-01-06
  • 가이드
  • 첫주

이 가이드에서는 비주얼 스튜디오에서 프로젝트를 빌드하고 실행하는 방법을 설명합니다.

  1. 파일 탐색기에서 비주얼 스튜디오 프로젝트가 있는 폴더로 이동합니다.

    예) C:\SampleProject

  2. sln 파일을 더블클릭해서 비주얼 스튜디오를 실행합니다.

    예) SampleProject.sln

  3. 코드를 작성합니다. 또 작성합니다. 계속 작성합니다. (응?)

  4. 준비가 된 것 같으면 상단 메뉴로 가서 빌드(Build) > 솔루션 빌드(Build Solution)를 선택합니다.

  5. 출력(output) 창이 아래에 나타날 거에요. 만약 오류(error)나 경고(warning)이 있다면 오류 목록(Error List)에 나올 거예요. 모두 고치세요. 그리고 이 단계를 반복하세요.

    빌드 성공한 경우

    오류가 있을 경우

  6. 컴파일이 끝나면 F5 또는 버튼을 눌러서 프로그램을 실행합니다. 실행은 될 텐데 아마 콘솔 창이 곧바로 사라질 겁니다.

  7. 이 문제를 해결하려면, main 함수 끝의 코드 영역 왼쪽에 좁고 길쭉한 부분을 눌러 중단점(breakpoint)을 추가하면 됩니다. 중단점은 빨간 원의 모양을 하고 있습니다.

  8. 프로그램을 다시 실행합니다. 결과를 봐야죠. F5 또는 버튼을 다시 누르면 콘솔 창이 바로 닫히지 않고 남아 있습니다.

관련 가이드

  • 비주얼 스튜디오를 설치하는 방법은 이 가이드를 참고해주세요.
  • 비주얼 스튜디오에서 프로젝트에 파일을 추가하는 방법은 이 가이드를 참고해주세요.
