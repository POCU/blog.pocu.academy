가이드: 비주얼스튜디오 설치

문서봇 2019-01-06
  • 첫주
  • 가이드

본 가이드는 비주얼 스튜디오 커뮤니티 2022 설치 방법을 설명합니다. 이 설치 방법은 POCU 아카데미의 과목을 수강하시는 분들을 기준으로 작성되었음을 미리 알려드립니다.
(※주의: 비주얼 스튜디오 코드가 아닙니다.)

  1. 비주얼 스튜디오 홈페이지에서 설치 파일을 다운로드합니다.

  2. 다운로드한 설치 파일을 실행합니다.

  3. 사용 약관 및 개인정보처리 방침 동의요청 팝업이 나오면 계속 버튼을 눌러 설치를 진행합니다.

  4. 설치 프로그램 위에 워크로드라는 팝업창이 뜰 겁니다. 과목별로 설치하는 워크로드가 조금씩 다릅니다. 아래 표를 보시고 해당 과목에 맞는 워크로드의 체크박스를 체크해 주세요.

    과목코드 과목명 워크로드
    COMP1500 실무 프로그래밍 입문 웹 및 클라우드 항목 > ASP.NET 및 웹 개발
    COMP1000 소프트웨어 공학용 수학 웹 및 클라우드 항목 > ASP.NET 및 웹 개발
    COMP2200 C 언매니지 프로그래밍 데스크톱 및 모바일 항목 > C++를 사용한 데스크톱 개발
    COMP2300 어셈블리 프로그래밍 데스크톱 및 모바일 항목 > C++를 사용한 데스크톱 개발
    COMP3200 C++ 언매니지 프로그래밍 데스크톱 및 모바일 항목 > C++를 사용한 데스크톱 개발

  5. 우측 하단의 설치 버튼을 누릅니다.

  6. 혹시라도 컴퓨터를 재부팅해야 된다는 메시지가 나오면 재부팅해주세요.

  7. 이제 비주얼 스튜디오를 실행합니다.

  8. 로그인 하라는 팝업 창이 등장할 겁니다. 강좌를 진행하는 데는 크게 필요가 없으니 나중에 로그인을 눌러 건너뛰세요.

  9. 비주얼 스튜디오 테마를 설정하는 팝업창이 나올 텐데요. 개발 설정은 일반으로 설정하시고 테마는 마음에 드시는 테마로 선택하고 Visual Studio 시작을 누릅니다.

간단하죠? 그럼 즐거운 강의되시길 바랍니다 :D
