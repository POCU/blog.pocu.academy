가이드: 비주얼 스튜디오에 새로운 파일을 추가하는 방법

문서봇 2019-01-07
  • 첫주
  • 가이드

이 가이드에서는 비주얼 스튜디오에 새로운 파일(.h 파일, .cpp 파일)을 추가하는 방법을 설명합니다.

비주얼 스튜디오를 설치하는 방법은 이 가이드를 참고해주세요.

  1. 파일 탐색기에서 비주얼 스튜디오 프로젝트가 있는 폴더로 이동합니다.

    예) C:\SampleProject

  2. sln 파일을 더블클릭해서 비주얼 스튜디오를 실행합니다.

    예) SampleProject.sln

  3. 솔루션이 열리면 솔루션 탐색기(Solution Explorer)를 찾으세요.

  4. 프로젝트 이름에 우클릭 > 추가(Add) > 새 항목(New Item)을 선택합니다.

  5. 헤더 파일을 먼저 만들어봅시다. Visual C++ > 헤더 파일(Header File)(.h)을 선택하고 파일 이름을 적절히 입력 후 추가(Add) 버튼을 누릅니다.

  6. C++ 파일을 이제 만들어봅시다. 같은 방식으로 C++ 파일(C++ File)(.cpp)를 선택하고 파일 이름을 적절히 입력 후 추가(Add) 버튼을 누릅니다.

  7. 솔루션 탐색기 창에서 파일들이 모두 있는지 확인합니다.
