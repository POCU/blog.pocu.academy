  • 가이드: 비주얼 스튜디오에 새로운 파일을 추가하는 방법

    이 가이드에서는 비주얼 스튜디오에 새로운 파일(.h 파일, .cpp 파일)을 추가하는 방법을 설명합니다. 비주얼 스튜디오를 설치하는 방법은 이 가이드를 참고해주세요. 파일 탐색기에서 비주얼 스튜디오 프로젝트가 있는 폴더로 이동합니다. 예) C:\SampleProject sln 파일을 더블클릭해서 비주얼 스튜디오를 실행합니다. 예) SampleProject.sln 솔루션이 열리면 솔루션 탐색기(Solution Explorer)를 찾으세요. 프로젝트 이름에 우클릭 > 추가(Add) > 새 항목(New Item)을 선택합니다. 헤더 파일을 먼저 만들어봅시다. Visual C++ > 헤더 파일(Header File)(.h)을 선택하고 파일 이름을 적절히 입력 후 추가(Add) 버튼을 누릅니다. C++ 파일을 이제 만들어봅시다. 같은 방식으로 C++ 파일(C++ File)(.cpp)를 선택하고 파일 이름을 적절히 입력 후 추가(Add) 버튼을 누릅니다. 솔루션 탐색기 창에서 파일들이 모두 있는지 확인합니다.

  • 가이드: 내 Git 저장소에 다른 유저를 초대하기

    POCU 아카데미에서 지원하는 Git 저장소 서비스는 아래와 같습니다. 깃허브(GitHub): 무료 깃랩(GitLab): 무료 따라서 본 가이드에서는 이 두 Git 저장소 서비스의 Git 저장소에서 다른 유저를 초대하는 방법을 설명합니다. 깃허브 다음은 깃허브의 Git 저장소에 다른 유저를 초대하는 방법입니다. 로그인 후 저장소 페이지로 갑니다. Settings > Collaborators를 선택 Add people 선택 후 초대할 사람의 Username을 빈칸에 입력 검색 결과에서 초대할 사람을 선택 Select a collaborator above를 누른 후 수락을 기다립니다. 깃랩 다음은 깃랩의 Git 저장소에 다른 유저를 초대하는 방법입니다. 로그인 후 저장소 페이지로 갑니다. 왼쪽 메뉴 모음에서 Manage > Members를 선택 Project members 페이지 우측 상단의 Invite members 버튼을 누릅니다. 팝업 창이 하나 뜰겁니다. 각각을 다음과 같이 설정합니다. Username or email address: 초대하려는 멤버의 Username 또는 이메일 주소를 적으면 바로 아래에 검색 결과가 뜹니다. 거기서 초대할 멤버를 선택 Select a role: Reporter를 선택 Access expiration date (optional): 필수가 아니라 설정할 필요 없습니다. 모두 입력했으면 Invite 버튼을 누릅니다. Project members 페이지 `Members`` 부분에서 추가된 멤버를 확인할 수 있습니다.

  • 가이드: 비주얼스튜디오 설치

    본 가이드는 비주얼 스튜디오 커뮤니티 2022 설치 방법을 설명합니다. 이 설치 방법은 POCU 아카데미의 과목을 수강하시는 분들을 기준으로 작성되었음을 미리 알려드립니다. (※주의: 비주얼 스튜디오 코드가 아닙니다.) 비주얼 스튜디오 홈페이지에서 설치 파일을 다운로드합니다. 다운로드한 설치 파일을 실행합니다. 사용 약관 및 개인정보처리 방침 동의요청 팝업이 나오면 계속 버튼을 눌러 설치를 진행합니다. 설치 프로그램 위에 워크로드라는 팝업창이 뜰 겁니다. 과목별로 설치하는 워크로드가 조금씩 다릅니다. 아래 표를 보시고 해당 과목에 맞는 워크로드의 체크박스를 체크해 주세요. 과목코드 과목명 워크로드 COMP1500 실무 프로그래밍 입문 웹 및 클라우드 항목 > ASP.NET 및 웹 개발 COMP1000 소프트웨어 공학용 수학 웹 및 클라우드 항목 > ASP.NET 및 웹 개발 COMP2200 C 언매니지 프로그래밍 데스크톱 및 모바일 항목 > C++를 사용한 데스크톱 개발 COMP2300 어셈블리 프로그래밍 데스크톱 및 모바일 항목 > C++를 사용한 데스크톱 개발 COMP3200 C++ 언매니지 프로그래밍 데스크톱 및 모바일 항목 > C++를 사용한 데스크톱 개발 우측 하단의 설치 버튼을 누릅니다. 혹시라도 컴퓨터를 재부팅해야 된다는 메시지가 나오면 재부팅해주세요. 이제 비주얼 스튜디오를 실행합니다. 로그인 하라는 팝업 창이 등장할 겁니다. 강좌를 진행하는 데는 크게 필요가 없으니 나중에 로그인을 눌러 건너뛰세요. 비주얼 스튜디오 테마를 설정하는 팝업창이 나올 텐데요. 개발 설정은 일반으로 설정하시고 테마는 마음에 드시는 테마로 선택하고 Visual Studio 시작을 누릅니다. 간단하죠? 그럼 즐거운 강의되시길 바랍니다 :D

  • 가이드: 소스트리 설치

    안녕하세요. POCU 아카데미 수강생 여러분 이 가이드는 Git GUI 툴인 소스트리(Sourcetree)를 설치하는 방법입니다. 소스트리 사이트에서 설치 파일을 다운로드하여 실행합니다. 계정 등록 화면이 나오면 건너뛰기를 선택합니다. 도구 설치 창이 나오면 Git 체크박스가 체크된 걸 확인하고, 고급옵션에서 기본적으로 줄 끝을 자동으로 처리하도록 설정(권장) 체크박스를 해제합니다. 이제 다음을 눌러서 설치를 마저 진행합니다 Author Name, Email Address를 입력(자동 입력되는 경우도 있음)하고 설치를 마무리합니다. 설치 완료 후, SSH키를 불러오시겠습니까? 라는 메시지가 뜨면 아니오를 누릅니다. 소스트리 같은 툴은 실무에서 필수로 쓰이니 이번 기회에 배워놓으시면 나중에 도움이 많이 될 겁니다! 그럼 즐거운 강의되시길 바랍니다 :D

  • 가이드: 웹 기반 비공개 Git 가입하고 저장소 만들기

    이 가이드에서는 웹 기반의 비공개 Git 저장소를 만드는 방법을 설명합니다. POCU 아카데미에서 지원하는 Git 저장소 서비스는 아래와 같습니다. 깃허브(GitHub): 무료 깃랩(GitLab): 무료 깃허브 첫 번째로 깃허브에서 비공개 Git 저장소를 만드는 방법을 안내하겠습니다. 깃허브 계정 만들기 깃허브에서 Git 저장소를 만드려면 회원 가입을 해야합니다. 깃허브 홈페이지에 들어가서 우측 상단의 Sign Up 버튼을 누릅니다. Username, Email address, Password를 입력하고, 퍼즐까지 풀고 Submit을 누릅니다. 입력한 Email 계정으로 온 launch code를 입력합니다. 간단한 설문조사를 하는 화면이 나오면 설문에 답을 하고 Continue를 누르거나 Skip personalization을 누릅니다. Plan을 고르는 화면이 나오면 Continue for free(무료)를 선택합니다. 비공개 저장소 만들기 회원 가입을 하셨다면, 이제 저장소를 만들 차례입니다. 깃허브 홈페이지에 로그인합니다. New 혹은 Create a new repository 버튼을 누릅니다. Repository name을 자유로이 입력합니다. Public과 Private 중 꼭 Private을 선택해야 됩니다. 깃랩 다음은 깃앱에서 비공개 Git 저장소를 만드는 방법입니다. 깃랩 계정 만들기 깃랩 역시 Git 저장소를 만들기 위해서는 회원 가입이 필요합니다. 깃랩 홈페이지에 갑니다. 오른쪽 상단 구석의 Sign in를 누릅니다. Registser now 링크를 누릅니다. 가입하는 방법은 두 가지가 있는데 편하신 방법을 따라 해 주세요. 소셜 로그인을 통해 가입하기 아래로 스크롤합니다. 원하는 소셜 로그인을 선택해서 진행합니다. 이 경우 로그인 후에 Settings > Account에서 Username을 확인하고, Settings > Password에서 Password를 만들어야 합니다. 직접 가입하기 프라이버시 보호 차원에서 Full name은 아무 이름이나 입력하셔도 됩니다. Username, Email, Password는 제대로 입력합니다. 아래 Register 버튼을 누릅니다. 화면에 나오는 설명에 따라 가입을 완료하세요. 비공개 저장소 만들기 회원 가입을 하셨다면, 이제 저장소를 만들 차례입니다. 깃랩 홈페이지에 로그인합니다. 우측 상단에 New project를 누릅니다. Create Blank project/repository 탭을 선택합니다. Project name을 입력합니다. 마음에 드는 이름을 넣어 주세요. Visibility Level은 꼭 Private을 선택해야 됩니다. Create project를 누릅니다.
